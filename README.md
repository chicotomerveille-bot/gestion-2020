# 🍟 Chips Gestion Pro

Logiciel de gestion complet pour une entreprise de production de chips.

## Fonctionnalités

- 📊 Tableau de bord avec statistiques en temps réel
- 👥 Gestion des clients et dettes
- 🏭 Suivi de la production journalière par employé
- 💰 Enregistrement des montants reçus
- 💸 Gestion des dépenses et salaires
- 📦 Gestion des stocks (entrées/sorties)
- 🛒 Suivi des commandes (Kanban)
- 📈 Finances et analyses PME
- 👷 Registre des employés
- 🗑️ Corbeille avec restauration (7 jours)
- 📥 Export CSV de toutes les données

## Déploiement

Ce projet est un fichier HTML statique unique.  
Toutes les données sont stockées **localement dans le navigateur** (localStorage).

## Déployer sur Vercel

1. Forkez ou importez ce dépôt sur GitHub
2. Connectez-vous sur [vercel.com](https://vercel.com)
3. Cliquez **"Add New Project"** → importez le dépôt GitHub
4. Vercel détecte automatiquement le projet statique
5. Cliquez **Deploy** — c'est tout !

## Utilisation locale

Téléchargez `index.html` et ouvrez-le directement dans votre navigateur.
